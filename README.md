# Delivery Driver Pay Calculator

A calculator that helps delivery drivers determine their true hourly wage after accounting for all expenses.

## Versions

- **US Version** (`index.html`) - Amazon Flex, FedEx, UPS, USPS - 2026 IRS mileage rate ($0.725/mile)
- **UK Version** (`uk.html`) - Amazon Flex, DPD, Evri, Royal Mail - 2026 HMRC mileage rate (45p/mile)

## Features

- Calculate true hourly pay after expenses
- Region-specific mileage rates (IRS for US, HMRC for UK)
- Currency and fuel units localized ($/gallon vs £/litre)
- Include fuel, depreciation, maintenance, and taxes (Self-Employment Tax / National Insurance)
- Visual breakdown of earnings vs expenses
- Comprehensive FAQ for delivery drivers
- 100% client-side - no data leaves your browser

## Tech Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Local Development

Just open `index.html` in your browser. No build step required.
